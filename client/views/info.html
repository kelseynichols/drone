How it works:

User

the user creates buildings and routes through our user friendly account page

Software

building deminsions and route details are stored and passed on to the drone

Hardware

the drone receives the data and flies the specified path

Create

Create a 3D model of your building by inputting custom length, width, and height.

Draw

Draw a route using Google Maps integration. Specifiy the height and desired GPS coordinates.

FLy!

Watch the drone fly the route you drew around the building you created. Cool, huh?
